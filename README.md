# UserMan

## Dependencies

- Angular Cli
- RESTful Server [https://github.com/siteslave/rest-server](https://github.com/siteslave/rest-server)
- Electron

# Installation

```
$ git clone https://github.com/siteslave/angular2-electron-kkh

$ cd angular2-electron-kkh

$ npm i
```

### For Web

```
$ ng serve
```

### For Desktop

```
$ npm run electron
```
